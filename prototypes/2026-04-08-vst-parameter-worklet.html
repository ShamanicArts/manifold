🔄 Bidirectional Sync Demo

This prototype demonstrates how the VST export system handles parameter synchronization:

  • Normalized values (0-1) are mapped to physical ranges using skew factors
  • Audio thread reads from std::atomic<float> (lock-free)
  • Message thread handles UI updates and host automation callbacks
  • View modes switch between compact (236×220) and split (472×220)
